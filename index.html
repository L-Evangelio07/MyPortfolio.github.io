About me

Building bridges between hardware and software.

I am a 3rd year Computer Engineering student at the University of St. La Salle Bacolod, with a strong foundation in embedded systems, electronics, networking, and software development. I am driven by a genuine interest in building systems that bridge the gap between hardware and software.

Through academic projects and hands-on coursework, I have developed practical experience in microcontrollers, robotics, circuit and PCB design, programming architecture, systems development, and software UI/UX design.

I am highly motivated and committed to contributing to an organization by applying my knowledge in troubleshooting, hardware integration, software development, and cross-functional technical collaboration — while continuing to grow as an engineer in an industry setting.

My work

Projects

Line Follower Robot

Designed and built an autonomous line-following robot using infrared sensors and a microcontroller to detect and track a predefined path in real time. The project involved sensor calibration, motor control logic, and circuit integration, demonstrating my ability to combine hardware and embedded programming into a functional autonomous system.

ArduinoC++/CSensors

Obstacle Avoidance Robot

Developed a robot capable of detecting and navigating around obstacles in real time using ultrasonic sensors and programmed decision logic. The system processes sensor data to make autonomous directional decisions, highlighting my hands-on experience in robotics, embedded systems, and sensor-based automation.

ESP32UltrasonicAutomation

Self-Designed PIC & Module

Designed a custom PIC module from the ground up, covering schematic design, PCB layout, component soldering, and validation using simulation tools such as Proteus and Multisim. This project reinforced my understanding of microcontroller interfacing, circuit fabrication, and hardware debugging.

PCB DesignProteusMultisim

Payroll System

Built a desktop-based payroll management system that automates employee compensation computation, deduction tracking, and report generation. The system streamlines payroll processing workflows and ensures accuracy in financial data handling, applying my knowledge in systems design and database management.

Stock Inventory System

Developed a cross-platform stock inventory system with both web and mobile app versions, allowing users to manage product records, monitor stock levels, and generate inventory reports. The project demonstrated my ability to build end-to-end solutions across multiple platforms using consistent data management logic.

Vacation Package Application

Designed and developed a mobile vacation booking application that allows users to browse, select, and book travel packages. The project involved UI/UX prototyping in Figma, followed by full implementation — showcasing my ability to translate user-centered designs into working mobile applications.

Coffee Shop Application

Designed and developed a mobile coffee shop application that allows users to browse the menu, customize orders, and complete in-app purchases with ease. The project involved UI/UX prototyping in Figma to establish a warm and inviting interface, followed by full mobile implementation — showcasing my ability to build user-centered applications with intuitive navigation and a polished visual identity.
